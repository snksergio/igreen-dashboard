Drawer
Transaction detail drawer — a 620px slide-in panel with header, metadata grid, tabs, and 4 content panels.
Drawer Header
The .dw-header contains a coin icon, title/subtitle wrap, and a close button. The icon uses an inline background color for the asset type.

Tabs
The .dw-tabs bar provides navigation between drawer panels. Active state uses a bottom border indicator. Badges show counts with .dw-tab-badge.

API
Drawer
Class Description Type Default
drawer-overlay Full-screen backdrop behind the drawer panel. layout hidden
drawer-overlay.open Shows the overlay with fade-in transition. state -
drawer Fixed 620px panel, slides from right edge. layout hidden
drawer.open Translates the panel into view. state -
dw-header Flex container for icon, title wrap, and close button. layout -
dw-coin-icon 48px square icon with rounded corners. element -
dw-meta Two-column metadata grid (120px + 1fr). layout -
dw-row Grid row using display: contents. layout -
dw-tabs Horizontal tab bar with border separators. navigation -
dw-tab Individual tab button. element inactive
dw-tab.active Active tab with bottom border indicator. state -
dw-tab-badge Count badge inside a tab. element -
dw-body Scrollable content area below tabs. layout -
dw-panel Tab panel container (hidden by default). layout hidden
dw-panel.active Visible tab panel. state -
dw-comment-box Comment input container with toolbar. element -
dw-comment Single comment with avatar and body. element -
dw-attachment File attachment row with icon and metadata. element -
dw-timeline Vertical timeline with line and dot indicators. layout -
dw-tl-dot.completed Green (primary) status dot. state -
dw-tl-dot.pending Yellow (warning) status dot. state -
dw-tl-dot.failed Red (destructive) status dot. state -
