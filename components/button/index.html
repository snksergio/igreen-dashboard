Button
Trigger actions and events. Composable with variants, sizes, icons, and states.
Variant
Set the button appearance with variant classes: btn--solid, btn--outline, btn--ghost, btn--destructive, btn--warning, or btn--info. Default has no modifier.
Size
Four sizes available: btn--xs, btn--sm, md (default), and btn--lg.
Icon
Use btn--icon for square icon-only buttons. Combine with any size.
Button With Icon
Icons can be added alongside text. Place the SVG before or after the label.
Loading
Add btn--loading to show a spinner and disable interaction.
Block
Use btn--block to make a button fill its container width.
Disabled
Set disabled attribute to prevent interaction.
Shape
Use btn--pill for pill-shaped buttons.
Composition
Combine multiple modifiers freely. Classes are composable.
