API
Class Type Description
.card Base Base card container with background, border-radius, padding, and hover shadow.
.card-head Sub Flex header row for title area and action buttons.
.card-title Sub Card heading text. 15px, weight 600.
.card-sub Sub Muted subtitle below the card title.
.card-actions Sub Flex container for action buttons in the card header.
.ca-btn Action 34px square icon button with border and background.
.more-dots Action 34px square overflow/more menu trigger. Ghost style.
.perf-stats Sub Flex row wrapper for stat sub-cards.
.pstat-card Sub Individual stat sub-card with muted background and hover accent.
.pstat-label Sub Uppercase label inside a stat card.
.pstat-val Sub Large numeric value inside a stat card. Tabular-nums enabled.
.pstat-val.green State Green (positive) color for the stat value.
.pstat-val.red State Red (negative/destructive) color for the stat value.
.pstat-pct Sub Muted percentage or unit label beside the stat value.
.perf-legend Sub Flex row wrapper for chart legend items.
.pl-item Sub Individual legend item with color swatch and label.
.pl-line Sub 14px wide, 2px tall color swatch in the legend.
.chart-area Sub Flex-grow container for chart content.
.donut-area Sub Flex-grow container for donut/pie chart. Min-height 120px.
.alloc-list Sub Wrapper for allocation list rows.
.al-row Sub Individual allocation row with top border separator.
.al-left Sub Left side of allocation row (pip + name).
.al-pip Sub 8px circular color indicator dot.
.al-name Sub Asset name text in the allocation row.
.al-right Sub Right side of allocation row (percentage + change badge).
.al-pct Sub Percentage value. Bold, tabular-nums, right-aligned.
.al-chg Sub Change badge with pill shape. Base class for up/down variants.
.al-chg.up State Positive change badge. Green text on green-10 background.
.al-chg.dn State Negative change badge. Red text on red-10 background.
