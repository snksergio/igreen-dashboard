Class Description Type Default
.pagination Container. Flexbox row with space-between alignment and a top border separator. layout required
.page-info Text element showing the current range and total count (e.g. "Showing 1-10 of 1,247"). text -
.page-btns Flex container for page buttons with a 4px gap. layout -
.pb Individual page button. 34px min-width/height, centered content. base required
.pb.active Highlights the current page with primary background and bold text. state false
