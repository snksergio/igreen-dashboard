The .kpi-badge component supports two directional variants: .up for positive change and .dn for negative change.

API
KPI Card
Class Description Type Default
kpi-grid Grid container for KPI cards. Lays out children in a 4-column grid. layout 4 columns
kpi-card Base card container. Displays content and sparkline side by side. base required
kpi-left Left content area containing label, value, and footer. structure required
kpi-label Metric label text displayed above the value. element -
kpi-value Primary metric value. Bold, large, tabular-numeric. element -
kpi-footer Footer row containing badge and sub-text. structure -
kpi-badge Percentage change badge. Requires a direction modifier. element -
kpi-badge.up Positive change badge. Green/primary color scheme. modifier -
kpi-badge.dn Negative change badge. Red/destructive color scheme. modifier -
kpi-sub Sub-text shown next to the badge (e.g. "vs last month"). element -
kpi-spark Right-aligned sparkline chart container. Fixed 100px width. element 100px wide
