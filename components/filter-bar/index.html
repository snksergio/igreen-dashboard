Top-level page filter bar with period presets, date range selector, filter tags, and action buttons.
Complete Filter Bar
Full .filter-bar composition with period presets, date range, filter tags, and action buttons.
Period:
Preset Pills
Period selector built with .presets container and .preset buttons. Add .active to mark the selected period.
Date Range Button
Use .date-range-btn with a calendar icon and date text. Hover state highlights the border and background.
Filter Tags
Removable filter chips using .ftag. Optionally include a close icon inside the tag.
Action Buttons
Icon-only actions with .icon-btn and labeled actions with .export-btn.
